**********************************************************************************************************************
Steps to execute the program
**********************************************************************************************************************

1. Extract the zip folder
2. Copy the contents i.e TestCase1 and TestCase2 folders on to your system.
3. Open cmd prompt on your system. Traverse to the location on the system where    you copied the files.
4. Run the command - "decrypt.exe" 
5. Enter the ciphertext to get the plaintext guess.

Please Note:
Our source code works only when the dictionaries are in the format as supplied in the folders(i.e in the format similar to the english_words.txt file)
